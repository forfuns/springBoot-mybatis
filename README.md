# springBoot-mybatis
a spring-boot with mybatis basic demo


## install

idea import maven project

## configure

程序主要配置文件
`src/resources/application.properties`

mybatis mapper
`src/resources/mybatis/*`




## build

`mvn package`

## build without tests

`mvn package -Dmaven.test.skip=true`

## run

`java -jar path/to/jar.jar`
